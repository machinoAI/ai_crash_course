# AI/ML Senior Engineer Crash Course

A focused crash course and revision of the full stack needed to crack senior AI/ML/Data Scientist interviews at MAANG/ MANGOS-level companies.

**Covers:** Python, DSA, Classical ML algorithms, ML System Design, HLD & LLD, Deep Learning, NLP, LLM Fine-tuning, and GenAI stacks (RAG, AI Agents, MCP).

**Goal:** Concise, interview-ready references and hands-on code — less theory, more signal.

**Structure:** Each topic lives in its own directory with runnable examples and key notes.
